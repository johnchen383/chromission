Command Action
add [ws] adds current tab to workspace [ws]
close [ws] closes the workspace [ws]
open [ws] opens the workspace [ws]
add-all [ws] adds all tabs of active window to workspace [ws]
remove [ws] removes current tab from workspace [ws]
delete [ws] deletes the workspace [ws]
list list the workspaces stored
list [ws] list the sites stored in the workspace [ws]
reset resets the browser window with a fresh tab
